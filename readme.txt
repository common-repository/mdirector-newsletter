Connect for free your WordPress with MDirector, the most advanced and simple marketing & SMS tool of the market.

== Description ==

Why integrate your WordPress with MDirector? You will boost traffic towards your web, by communicating with your database the latest posts or contents you have published through effective email marketing campaigns.



== Installation ==

1. Install the plugin like you always install plugins, either by uploading it via FTP or by using the "Add Plugin" function of WordPress.
2. Activate the plugin at the plugin administration page
3. Configure: Open the plugin configuration page, which is located on the sidebar as "MDirector" and customize settings.
4. The plugin will automatically send your newsletter to all your registered users, so there is nothing more to do :)
